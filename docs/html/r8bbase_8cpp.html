Detailed Description

C++ file that should be compiled and included into your application.

This is a single library file that should be compiled and included into the project that uses the "r8brain-free-src" sample rate converter. This file defines several global static objects used by the library.

You may also need to include to your project: the "Kernel32" library (on Windows) and the "pthread" library on Mac OS X and Linux.
